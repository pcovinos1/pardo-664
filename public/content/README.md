Flujo:
1. Desde el administrador de la app, presiona "Descargar project.json para GitHub".
2. Reemplaza este archivo en public/content/project.json.
3. Haz commit y push en GitHub Desktop.
4. GitHub Pages publicara la nueva version.
5. Las tablets sincronizaran el contenido cuando vuelvan a tener internet.
